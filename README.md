This repository contains the implementation of our novel swarm-optimized hybrid frameworks for cyber-attack detection, combining meta-heuristic Artificial Bee Colony (ABC) algorithm with machine learning (KNN) and deep learning (GRU) models to create efficient and accurate Intrusion Detection Systems (IDS).
We developed two innovative hybrid architectures:

1. KNN+Bee Framework
ABC-automated feature selection from network traffic data

Optimal KNN parameter tuning through swarm intelligence

Lightweight yet accurate classification model

2. GRU+Bee Framework
ABC-optimized GRU architecture and hyperparameters

Enhanced temporal pattern recognition for sequential attacks

Balanced performance-complexity trade-off
